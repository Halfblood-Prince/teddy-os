# Assets

This directory will contain original Teddy-OS assets only.

Current guidance:

- keep all branding original to Teddy-OS
- avoid copied logos, icons, fonts, or exact vendor UI artwork
- keep themes simple and framebuffer-friendly for the current software renderer

Planned contents:

- wallpapers
- icon sets
- fonts
- color themes
- release manifests
